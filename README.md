# Qt Project Manager

Кроссплатформенное приложение для управления проектами на C++ с использованием фреймворка Qt.

## Требования

- Qt 5.15 или выше (или Qt 6)
- CMake 3.16 или выше
- Компилятор C++ с поддержкой C++17

## Структура проекта

```
/workspace
├── CMakeLists.txt              # Главный файл сборки CMake
├── README.md                   # Этот файл
├── src/                        # Исходный код
│   ├── main.cpp                # Точка входа приложения
│   ├── mainwindow.h/.cpp       # Главное окно приложения
│   ├── welcomescreen.h/.cpp    # Приветственный экран
│   ├── projectselectionscreen.h/.cpp  # Экран выбора проекта
│   └── projectcreator.h/.cpp   # Класс создания проектов
├── ui/                         # UI файлы (для будущего расширения)
└── resources/                  # Ресурсы приложения
```

## Функционал

1. **Приветственное окно** - начальная страница с кнопкой "Начать работу"
2. **Выбор проекта** - список шаблонов проектов (можно дополнять)
3. **Создание проекта** - генерация структуры проекта на основе CMake шаблона

## Сборка

```bash
mkdir build
cd build
cmake ..
make
```

## Запуск

```bash
./build/QtProjectManager
```

## Шаблоны проектов

В текущей версии доступны:
- Qt Widgets Application
- Console Application  
- Empty Project

При создании проекта генерируются:
- CMakeLists.txt для собранного проекта
- src/main.cpp с базовым кодом
- include/<projectname>.h с заголовком класса

## Кроссплатформенность

Приложение использует стандартные компоненты Qt и CMake, что обеспечивает работу на:
- Windows
- Linux
- macOS

## Примечания

- Не используется Python (только C++ и Qt)
- Проект открыт для расширения новыми функциями
